Installation Android

1
Scannez le QR code avec votre caméra ou application QR. Aucune connexion WiFi spécifique requise — n'importe quelle connexion internet fonctionne.
2
Le navigateur télécharge l'APK directement depuis internet.
3
Si demandé par Android, autorisez l'installation depuis Sources inconnues.
4
Ouvrez l'application et suivez les étapes : Inscription → Code d'activation → Code appareil.
