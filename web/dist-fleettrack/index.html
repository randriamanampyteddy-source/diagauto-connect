🔐 Administration

La gestion des licences et des clients est effectuée via l'application FleetTrack Admin (fichier séparé), réservée à l'administrateur système.

⚠️ Réinitialisation

Cette action supprime toutes les données de l'application : véhicules, trajets, carburant, routes personnalisées et prix du litre.

⚠️ Cette action est irréversible. Les données ne pourront pas être récupérées.
