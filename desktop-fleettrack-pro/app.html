🔔 Configuration ntfy.sh (notifications push)

ntfy.sh envoie des notifications sur votre téléphone quand un client se connecte ou fait une demande. Gratuit, aucun compte requis.

  1. Installez ntfy sur votre téléphone (Android / iOS) — ntfy.sh
  2. Choisissez un nom de topic unique ci-dessous (ex: fleettrack-rakoto-2024)
  3. Dans ntfy app → Ajouter un abonnement → entrez ce même topic
  4. Copiez ce topic dans les apps clients (tab Config de chaque app client)
📱 Numéro WhatsApp du client à contacter

Entrez le numéro du client avant de cliquer "Envoyer via WhatsApp" dans l'onglet Demandes.

🔥 Configuration Firebase (notifications temps réel)

Entrez votre config Firebase ci-dessous. Elle est sauvegardée localement — inutile de modifier le code.

  1. Allez sur console.firebase.google.com → Créer un projet
  2. Realtime Database → Créer une base de données (mode test)
  3. Règles → { "rules": { ".read": true, ".write": true } }
  4. Paramètres du projet → Vos applications → SDK → Copiez les valeurs ci-dessous
